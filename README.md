
# Clerk Next.js Quickstart for App Router

![Next.js Quickstart for App Router](./public/readme-hero.png)

This quickstart shows how to use [Clerk](https://clerk.com/?utm_source=DevRel&utm_medium=docs&utm_campaign=templates&utm_content=10-24-2023&utm_term=clerk-nextjs-app-quickstart) in a Next.js application.

Clerk is a developer-first authentication and user management solution. It provides pre-built React components and hooks for login, signup, user profile, and organization management. Clerk is designed to be easy to use and customize, and can be dropped into any React or Next.js application.

This quickstart allows you to get started with Clerk and Next.js in a matter of minutes, and includes:

- Next.js App Router install
- Auth flow using Clerk's Account Portal
- Finished application that matches our quickstart guide.

## Running the template

```bash
git clone https://github.com/clerkinc/clerk-next-app
```

To run the example locally you need to:

1. Sign up for a Clerk account at [https://clerk.com](https://clerk.com/?utm_source=DevRel&utm_medium=docs&utm_campaign=templates&utm_content=10-24-2023&utm_term=clerk-nextjs-app-quickstart).
2. Go to [Clerk's dashboard](https://dashboard.clerk.com?utm_source=github&utm_medium=template_repos&utm_campaign=nextjs_template) and create an application.
3. Set the required Clerk environment variables as shown in [the example env file](./.env.local.example).
4. `npm install` the required dependencies.
5. `npm run dev` to launch the development server.
